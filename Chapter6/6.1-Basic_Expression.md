# 6.1 概率论与统计学基本表达式


表6-1，列举了在概率论与统计学中最常用到的表达式，如期望、方差等。

表6-1 概率论与统计学基本表达式

| 公式                 | 含义             |
|----------------------|------------------|
| P(condition)         | 概率             |
| E(expression)        | 期望             |
| variance(expression) | 方差             |
| std(expression)      | 标准差           |
| density(expression)  | 概率密度函数     |
| sample(expression)   | 创建一个随机变量 |
| where(condition)     | 条件为真         |
| cdf                  | 累积分布函数     |
| skewness             | 偏态系数         |

这些函数位于sympy.stats中，使用下列代码，可将其一次导入
